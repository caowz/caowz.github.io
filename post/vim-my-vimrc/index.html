Vim

自从我大四下学期接触到了vim之后，便深深的爱上了这个文本编辑器，其功能的强大这里就不 做过多的介绍了。下面简单介绍一下如何在Ubuntu系统中安装及配置vim。Ubuntu系统通常会自带 vi，vim是在vi基础上的升级版本，对于一台没有安装vim的Ubuntu系统的电脑，在命令行输入 下列命令即可

1

sudo apt-get install vim-gnome

" set dycv
 " look up current word under cursor
 function! SearchWord()
     let expr = '!ydcv -s ' .expand("<cword>")
     exec expr
 endfunction
        
 " translate selected text
 function! SearchWord_v(type, ...)
     let sel_save = &selection
     let &selection = "inclusive"
     let reg_save = @@
            
     if a:0
         silent exe "normal! `<" . a:type . "`>y"
     elseif a:type == 'line'
         silent exe "normal! '[V']y"
     elseif a:type == 'block'
         silent exe "normal! `[\<C-V>`]y"
     else
         silent exe "normal! `[v`]y"
     endif
             
     let word = @@
     let expr = '!ydcv "' . word . '"'
     exec expr
            
     let &selection = sel_save
     let @@ = reg_save
 endfunction
            
 nnoremap <Leader>d :call SearchWord()<CR>
 vnoremap <Leader>d :<C-U>call SearchWord_v(visualmode(), 1)<cr>

每个人都有不同的习惯，在我的vim配置中，比较推荐的是在INSERT模式下设置光标的移动。 很多情况时我需要在INSERT模式中移动光标，如果不进行 设置，我需要先切换到NORMAL 模式下进行移动，但是这样就会变得比较麻烦，因此，我在配置文件中加入了如下命令

1
2
3
4

imap <C-H> <left>
imap <C-J> <up>
imap <C-K> <down>
imap <C-L> <right>

这样，就可以在INSERT模式下使用Ctrl+[h j k l]来进行光标的移动。

不过，尽管Vim可以通过配置使其具有很强的个性化，但是并不推荐大家进行太多的个性化配置， 否则换一个没有进行过配置的电脑可能就会非常的不习惯。
