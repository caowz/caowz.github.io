建立自己的博客可以对知识进行总结与分享，本文主要是介绍如何使用Hugo来搭建个人的博客。本文主要是对自己搭建过程的一个记录，因此可能不是非常的完整。

1. Ubuntu环境

本文主要是介绍的在Ubuntu16.04环境下搭建博客。在这里，还需要先安装Git等程序。这些软件的安装教程较多，因此这里不进行过多的介绍。

2. Hugo静态网站生成器

目前比较主流的静态博客网站生成器主要有Hexo和Hugo。本文主要是介绍如何使用Hugo的静态网站生成器，Hugo是由Go语言实现的静态网站生成器。具有简单、易用、高效、易扩展、快速部署等特点。 在网站上可以下载Hugo的发行版，这里就不过多的进行介绍了。

3. Hugo博客主题

Hugo目前已经提供了非常多的主题，我在这里采用的是even主题。关于该主题的介绍可以参考其文档。
